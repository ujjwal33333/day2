Beginning with Machine Learning, as we know the main aspect of machine learning includes pre-processing which I guarantee will take 80% of your time training a model.So yes, I learned quite a stuff about machine learning which revolved around struggling with Numpy, Pandas, Scipy, Matplotlib and not to forget Scikit-learn as well.

So beginning with Numpy, it is one of the most widely used python library for numeric python. Numpy guarantees great execution speed comparing it with python standard libraries. It comes with a great number of built-in functions. Advantages of using numpy with python are array-oriented computation, efficiently implemented multidimensional array, designed for scientific computation. So in order to install numpy you have to install it using pip command “pip install NumPy” or it comes pre-installed with conda as well.

Next comes Scipy, which depends on Numpy, and provides convenient and fast N-dimensional array manipulation. The scipy library is built to work with Numpy arrays, and provides many user-friendly and efficient numerical routines like for integration and optimization. This library consisted of hell-lot of mathematical functions so I restricted myself for now to learn some specific functions which included finding determinant, inverse, Singular Value Decomposition(SVD) and interpolate my data. Moreover it consisted of various statistical functions as well such as plotting histograms, creating a list of random values obeying various statistical function and to be more apt obeying various probability density functions like Guassian, Binomial, Bernoulli, Poission and many more.

And now comes matplotlib, and you guessed it right this is the library we use for plotting our data, it supports all the basic matlab functions and believe me in ML you will need this library a lot not in actual training of your model but to see if the training of model is going as you wanted or not.

Scikit-Learn is a vast library I haven’t explored it much yet, but it is the library where you begin actual training of data what we learned earlier was for preprocessing and using this library you actually train your data.

Next I looked over Web-Development and App-Development being my field of interests, I was quite fascinated to learn how webpages are designed using CSS, and creating an android app is something which I wanted to learn from a long time. While learning CSS I made a blog of myself, yes you got that write the one you are reading, so yes it was quite a fun to learn web-dev as well.I also explored App Developlment in android which turned pretty smooth by now, I realised that I need a hands-on with Java to excel in Android.

And last but not the least I had a walk-through with Hardware stuff as well, where I learned the basics of Ardruino, and some transmission protocols like i2C and UART. By now I also have an idea of what Master-Slave is in the world of Ardruino, and believe me it is the thing which everyone struggles to learn by heart, even I am not a gem in it yet.
